# IoT-Based Digital Notice Board

## Overview
This project is an IoT-based digital notice board that allows users to send notices remotely and display them on a screen using a microcontroller and internet connectivity.

## Features
- Real-time notice updates
- Wireless communication
- Remote access to send messages
- IoT-based display system

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Firebase
- Hardware: ESP8266
- Communication: WiFi

## How It Works
1. User sends notice through web interface
2. Data is stored in Firebase
3. ESP8266 fetches the data
4. Notice is displayed on the screen

## Project Structure
arduino_code/ → ESP8266 code
public/ → frontend files
firebase.json → Firebase config
